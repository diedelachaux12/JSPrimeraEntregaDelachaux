Se realizó un simulador de carrito utilizando el metodo push de array visto en la clase del 13/9 para mejorar el simulador.
Al iniciar el script se pregunta al usuario el nombre del producto a agregar, y luego el precio mediante otro prompt. Una vez indicado nombre y precio, se agrega a un array que va conteniendo los precios, y mediante un nuevo prompt se pregunta si se desea seguir agregando. En caso de que sí, solicitará nuevamente nombre y precio, en caso de que no, indicaráel valor final del carrito.
